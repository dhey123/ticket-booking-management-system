# ticket-booking-management-system
Web-based information and ticket booking system developed as a final thesis project using CodeIgniter 3 and MySQL.
## Features
- Online ticket booking
- Payment confirmation
- Visitor data management
- Ticket management
- Admin dashboard
- Reporting
- Gallery management
- Facility management

## Tech Stack

- CodeIgniter 3
- MySQL
- HTML
- CSS
- JavaScript
- Bootstrap
